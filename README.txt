CarDealerTest  https://github.com/JeffRabe/CarDealerTest

This project is a submission for a DataXu Coding test.
The specification is included at the project root level
in that attached .docx file.

The dx.cardealer.entity.CarWarehouse class represents the 
car dealer warehouse.  Cars and the related data structures
are contained in the dx.cardealaer.entity.car package.

The test source folder contains two JUnit Test Case classes
for the Car class and Car Warehouse class.
